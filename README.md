# 🧠 Quiz App

An interactive and responsive quiz application built using **HTML, CSS, and vanilla JavaScript**.  
This app allows users to test their knowledge with multiple-choice questions and view a score at the end.

## 🚀 Live Demo

👉 [Try the Quiz](https://yrb1.github.io/quiz/)

## 🌟 Features

- ❓ Multiple-choice questions  
- ⚡ Instant feedback once all questions are answered  
- 📊 Score display at the end  
- 🔄 Option to restart quiz  
- 📱 Responsive layout for all device sizes  
- 🎯 Clean and user-friendly interface  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
